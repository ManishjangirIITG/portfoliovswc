Patreon Clone

Welcome to my Patreon Clone project, a comprehensive platform built with HTML, CSS, JavaScript, React, MongoDB, Express, Node.js, Next.js, Next Auth, and UPI payment integration. This project aims to replicate the functionality of Patreon, offering creators a platform to connect with patrons and receive support for their work.

Key Features:

  • Creator-Patron Connection: Allows creators to share their work and connect with patrons who support them financially.
  • User Authentication: Implements Next Auth for secure user authentication and authorization.
  • UPI Payment Integration: Enables seamless payment transactions using UPI (Unified Payments Interface).
  • Data Persistence: Utilizes MongoDB to store user data, subscription details, and payment information.
  • Server-Side Rendering: Leveraging Next.js for efficient server-side rendering, providing a smooth user experience.
Explore my Patreon Clone to experience a platform where creators thrive and patrons support their favorite content creators!
